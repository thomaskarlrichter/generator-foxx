# Generator-foxx

A generator for Foxx applications based on Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-foxx`
- Run: `yo foxx`
